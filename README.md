# MUBL — Mirzo Ulugh Beg's Legacy

Student-led engineering and research community site built with Next.js (App Router), TypeScript, TailwindCSS, and light Framer Motion.

## Getting Started

1. Install dependencies
   ```bash
   npm install
   ```

2. Run the development server
   ```bash
   npm run dev
   ```
   Then open http://localhost:3000 in your browser.

3. Lint the project
   ```bash
   npm run lint
   ```

## Project Structure

- `app/` — Routes and page components
- `components/` — Shared UI components (navigation, cards, section headers)
- `public/` — Static assets and metadata placeholders
- `tailwind.config.ts` — Tailwind theming and scanning paths

## Design Notes
- Calm, research-oriented aesthetic with muted gradients and rounded cards
- Minimal animations for hover and entrances
- Navigation: Home, About, Projects, Events, Blog, Join Community
